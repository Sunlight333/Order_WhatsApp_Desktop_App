## рҹ”„ Complete User Flow Examples

### Flow 1: Create New Order
```
Login вҶ’ Order List вҶ’ Click "Create Order" вҶ’ Fill Form вҶ’ Submit вҶ’ 
Success Toast вҶ’ Order Detail Page
```

### Flow 2: Update Order Status
```
Order List вҶ’ Click Order вҶ’ Order Detail вҶ’ Click "Update Status" вҶ’ 
Select Status вҶ’ Confirm Modal вҶ’ Update API вҶ’ Success Toast вҶ’ 
Status Color Changes
```

### Flow 3: Notify Customer via WhatsApp
```
Order List/Detail вҶ’ Click Phone Number вҶ’ WhatsApp Opens вҶ’ 
Confirm Dialog вҶ’ Update Status вҶ’ Success Toast вҶ’ Status Green
```

### Flow 4: Search Orders
```
Order List вҶ’ Type in Search вҶ’ Debounced Search API вҶ’ 
Results Update вҶ’ Click Result вҶ’ Order Detail
```

### Flow 5: Delete User (Super Admin)
```
Users List вҶ’ Click Delete вҶ’ Confirmation Modal вҶ’ Confirm вҶ’ 
Delete API вҶ’ Success Toast вҶ’ User Removed from List
```
